## PRINTING NOTES

**Recommended specs:**
- Size: 4×6" postcard (most economical, fits mailboxes)
- Paper: 16pt cardstock, matte finish (premium feel, reads well in daylight)
- Print quantity to start: 250–500 per design, test 2–3 flyers first
- Double-sided: front = headline + visual, back = full copy + QR code

**Cheap printing options:**
- GotPrint.com — best price/quality for postcards
- Canva Print — design + print in one place
- FedEx Office — good for quick local prints to test

**QR code:**
Generate a free QR code at qr-code-generator.com pointing to:  
`https://apps.apple.com/app/id6778844291`

**Which flyers to start with:**
1. **Flyer 3 (The Math Flyer)** — price transparency builds trust fast
2. **Flyer 2 (Dress for the Job)** — strong for young professional neighborhoods
3. **Flyer 7 (The College Flyer)** — if near a university

Test those three first. See which drives the most App Store clicks, then print more of the winner.
